Module: elementrotator

The ElementRotator widget provides a simple mechanism for cycling through a set of elements with fade transitions. It is intended for elements of the same size, e.g. banner images. The widget will optionally sync transitions with the visitor's clock and/or persistently keep track of the currently shown element, so that the transitions are uniform even if the visitor reloads the page or navigates to another page with the same widget.
