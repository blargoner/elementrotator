Configuration Attributes

clock_sync - Boolean

Synchronize transitions with user's clock.
Default Value: false

cookie_name - String

Cookie name
Default Value: ElementRotator

cookie_persist - Boolean

Keep track of currently shown element with cookie
Default Value: false

cookie_settings - Object

Cookie settings (domain, path, etc.)
Default Value: { path: '/' }

fade_duration - Number

Duration of fade transition, in seconds
Default Value: 1

show_duration - Number

Duration each element is shown, in seconds
Default Value: 10
